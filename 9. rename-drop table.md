# MySQL Rename Table
### Syntax
```angular2html
 RENAME old_table TO new_table;  
```

### Example
```angular2html
RENAME TABLE old_tab1 TO new_tab1,  
             old_tab2 TO new_tab2, old_tab3 TO new_tab3;   
```

# MySQL DROP Table
### Syntax
```angular2html
mysql> DROP TABLE  table_name;  
OR,  
mysql> DROP TABLE  schema_name.table_name;  
```
### Example
```angular2html
 DROP TABLE  orders;    
```
